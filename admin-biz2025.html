Google Sheets 設定ガイド

このページは管理者向けです。Google Sheetsと連携することで、スタッフがスプレッドシートを編集するだけでダッシュボードに反映されます。

📖 設定手順
1
Google Sheetsを新規作成する
Google Driveで新しいスプレッドシートを作成します。
シートのタブを7つに分けて、各カテゴリのデータを入力してください。
1行目が必ず列名（ヘッダー）になるようにしてください。
2
シートをウェブに公開する
Googleスプレッドシートで：
ファイル → 共有 → ウェブに公開 → 対象シートを選択 → カンマ区切りの値（.csv） → 公開

発行されたURLが CSV URL です。カテゴリごとに繰り返してください。
3
dashboard-biz2025.html にURLを貼り付ける
dashboard-biz2025.html をテキストエディタで開き、
各カテゴリの sheetUrl: "", の "" の中にURLを貼り付けてください。
例：sheetUrl: "https://docs.google.com/spreadsheets/d/xxxxx/export?format=csv&gid=0",
4
ファイルをGitHubにプッシュ → GitHub Pagesで確認
変更をGitにコミットして git push してください。
GitHub Pagesのデプロイが完了したらダッシュボードURLで確認できます。
⚠️ 注意：共有設定について スプレッドシート自体は「リンクを知っている全員が閲覧可能」に設定してください。
編集権限は付与する必要はありません。スタッフはGoogleアカウントでシートを開いて編集してください。
📋 シートを開いて編集

URLが設定済みの場合はボタンからシートを直接開けます。未設定の場合はボタンが無効です。
